05Berechnung▼

Pro Tabelle eine Karte: aus Excel kopieren und in eine Karte einfügen (Strg + V). Farben & Formatierung bleiben erhalten. Für eine weitere Tabelle unten auf ＋ Weitere Tabelle. Mehrere auf einmal kopierte Tabellen werden automatisch auf eigene Karten verteilt.

pt

Zielgröße für Tabellen. Alle Spalten bleiben immer sichtbar — ist eine Tabelle breiter als die Seite, wird sie auf Seitenbreite eingepasst (die Schrift wird dabei automatisch kleiner). Sehr breite Tabellen (viele Spalten) erreichen so ca. 6–7 pt; größer geht nur mit weniger Spalten.
